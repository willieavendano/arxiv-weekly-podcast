# arXiv Weekly — AI Research Briefing Podcast

🎙️ 20-30 minute weekly audio briefings on the most interesting AI/ML research across 20 arXiv categories.

**Subscribe in Overcast or any podcast app:**
```
https://willieavendano.github.io/arxiv-weekly-podcast/feed.xml
```

## What It Covers

| Tier | Categories |
|------|-----------|
| Core AI/ML + OR | cs.AI, cs.CL, cs.LG, cs.CV, stat.ML, cs.MA |
| Directly Relevant | math.OC, cs.HC, cs.CR |
| Range & Curiosity | quant-ph, q-bio.NC, cs.IT, math.ST, math.CT, cs.GT |
| Breadth | math.PR, physics.comp-ph, cs.NE, cs.SE, econ.GN, cs.RO |

## Episode Format

Each ~25-minute episode covers 5-8 papers:

- **📄 Per paper (3-4 min)**: What problem it solves, key contribution, why it matters to Willie's world
- **📊 Trends (2 min)**: Patterns across the week's research
- **👋 Outro**: Full vault note reference

## How It Works

Every Monday, a Hermes cron job:
1. Searches 20 arXiv categories for the week's papers
2. Enriches with Semantic Scholar impact data
3. Filters to the top 5-8 most interesting
4. Synthesizes a vault briefing
5. Generates a spoken script
6. Produces audio via ElevenLabs Lily voice
7. Updates this RSS feed

## Voice

Hermes-written, Lily-voiced. Conversational, personalized — not a cold academic summary.
